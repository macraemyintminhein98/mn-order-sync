📧 Weekly Workflow — MN Homes Orders

Ashley Myers (MN Custom Homes) emails weekly order tables, titled by week (e.g. "5/31 Week"). Follow these steps every week:

1
Open the email in the Outlook app

Use the desktop Outlook app, not the browser — the browser version cuts off tables when printing.

2
Save the whole thread as PDF

With the email open, press Ctrl+P → printer: Microsoft Print to PDF (or Save as PDF) → Print/Save. This captures the entire thread including urgent requests buried in replies.

3
Name it with the week number

Use the week from the subject line, e.g. 5-31 Week.pdf, and save it on the SignPros server under:
Digitech products → MN homes → new orders

4
Upload here

Go to the Upload tab → choose 🚀 Full Email — Auto-Detect → drop the PDF. The AI scans every page and pulls out the On-Market Riders table automatically (all other tables in the email are ignored — riders are the only thing we track here).

5
Review, fix, save

Check the extracted rows against the email — click any cell to correct it, click a Cancelled cell to toggle. Then Save to Database. The whole team sees it instantly.

💬 Team Notes

Every order has a 💬 notes button in the Orders tab. Use it to log who took the order, install issues, date changes, or anything the next person needs to know. Notes are shared — if Kelly adds a note, Dick sees it. Your name is asked once and remembered on this device.

✏️ Daily Use

✓
Mark work done

Orders tab → click the Date Completed / Date Removed cell → type the date. Status flips to Completed automatically.

✓
Search everything

The Orders tab lists every rider — search by address, PM, or order #, filter by PM or status, sort by date or name.

✓
Friday backup

Settings → Backup All Data. Takes 2 seconds, saves the whole database as one file.

⚠️ Tips

• If extraction misses rows, crop a screenshot of just that table and upload it under its specific category instead.
• Duplicate order numbers are skipped automatically when uploading multi-page threads.
• If the AI is rate-limited (busy day), wait a minute and retry — it auto-falls back to other models.
• Pricing for all MN items (riders, barricades, safety signs, panels) lives in the Price Book tab with the quote builder.
